# 财务智能助手

专为财务部门设计的智能助手，能够分析财务Excel数据，进行费用审核，并生成专业的PDF财务报告。

## 核心能力

1. **财务数据分析** - 分析收支、利润、成本结构，计算关键财务指标
2. **费用报销审核** - 审核报销单据，检查合规性，统计报销情况
3. **财务文书生成** - 生成财务分析报告、预算执行报告、月度/季度财务简报

## 关键财务指标

skill会自动计算以下常用指标：

### 盈利能力指标
| 指标 | 公式 | 说明 |
|------|------|------|
| 毛利率 | (收入-成本)/收入×100% | 核心盈利能力 |
| 净利率 | 净利润/收入×100% | 最终盈利水平 |
| 费用率 | 总费用/收入×100% | 费用控制能力 |

### 预算执行指标
| 指标 | 公式 | 说明 |
|------|------|------|
| 预算执行率 | 实际支出/预算金额×100% | 预算使用情况 |
| 预算偏差率 | (实际-预算)/预算×100% | 超支或节约程度 |
| 预算剩余率 | 剩余预算/总预算×100% | 剩余可用空间 |

### 费用结构指标
| 指标 | 公式 | 说明 |
|------|------|------|
| 人力成本占比 | 人力成本/总成本×100% | 人力投入比重 |
| 管理费用占比 | 管理费用/总费用×100% | 管理成本比重 |
| 销售费用占比 | 销售费用/总费用×100% | 销售投入比重 |

### 报销审核指标
| 指标 | 公式 | 说明 |
|------|------|------|
| 人均报销额 | 总报销金额/报销人数 | 人均费用水平 |
| 审核通过率 | 通过数量/总数量×100% | 报销合规程度 |
| 平均审核周期 | 总审核天数/报销单数 | 审核效率 |

## 工作流程

### 第一步：理解数据类型

用户可能提供以下类型的财务Excel：

**收支明细类型**（包含但不限于）：
- 日期、摘要、收入、支出、余额
- 科目、部门、项目、金额
- 收款/付款方、发票号、备注

**费用报销类型**（包含但不限于）：
- 报销人、部门、报销日期
- 费用类型（差旅、招待、办公、交通等）
- 金额、发票数、审批状态
- 报销事由、附件说明

**预算数据类型**（包含但不限于）：
- 预算科目、预算金额、实际金额
- 部门、项目、时间周期
- 调整金额、剩余金额

### 第二步：解析Excel数据

使用内置脚本解析财务Excel：

```bash
python "<skill-path>/scripts/finance_excel_parser.py" "<excel文件路径>" --output "<输出json路径>"
```

脚本会输出JSON文件，包含：
- `data`: 解析后的数据列表
- `data_type`: 识别的数据类型（income_expense / reimbursement / budget / unknown）
- `summary`: 自动生成的财务摘要
- `indicators`: 计算好的财务指标

### 第三步：分析数据并生成内容

根据数据类型执行不同分析：

**收支数据分析**：
- 收入/支出趋势图（按月/季度）
- 收支结构饼图
- 毛利率、净利率计算
- 同比/环比变化分析
- 异常值检测（大额支出提醒）

**费用报销审核**：
- 按部门/人员统计报销金额
- 费用类型分布
- 超标报销标记（如单笔超5000元）
- 高频报销人员预警
- 审核状态统计

**预算执行分析**：
- 预算vs实际对比柱状图
- 各部门预算执行率
- 超支项目标红预警
- 节约项目标绿
- 预算剩余分析

### 第四步：生成PDF报告

使用内置脚本生成PDF：

```bash
python "<skill-path>/scripts/finance_pdf_generator.py" \
  --title "报告标题" \
  --content "<内容json文件路径>" \
  --output "<输出pdf路径>" \
  --charts "<图表数据json路径>"
```

内容JSON格式：
```json
{
  "report_type": "monthly_finance",
  "period": "2025年3月",
  "sections": [
    {
      "title": "章节标题",
      "content": "分析文字内容",
      "type": "text"
    },
    {
      "title": "财务指标汇总",
      "data": [["指标", "数值", "同比"], ["毛利率", "35.2%", "+2.1%"]],
      "type": "table"
    },
    {
      "title": "收支趋势",
      "type": "chart",
      "chart_title": "月度收支趋势"
    }
  ],
  "indicators": {
    "total_income": 1500000,
    "total_expense": 980000,
    "net_profit": 520000,
    "profit_rate": "34.67%"
  }
}
```

## 报告模板

### 月度财务报告结构

**封面**
- 报告标题（XX公司X月财务分析报告）
- 报告期间
- 生成日期

**执行摘要**
- 本月核心数据（收入、支出、利润）
- 关键指标变化（同比/环比）
- 重要发现和风险提示

**收支分析**
- 收入构成分析
- 支出结构分析
- 收支趋势图表

**预算执行**
- 预算完成情况
- 各部门预算对比
- 超支/节约分析

**费用明细**
- 费用类型分布
- 部门费用排名
- 异常费用说明

**建议与展望**
- 财务风险提示
- 改进建议
- 下期预算建议

## 费用审核规则

内置以下审核规则（可自定义）：

| 费用类型 | 单笔限额 | 月度限额 | 需附件 |
|----------|----------|----------|--------|
| 差旅费 | 5000元 | 20000元 | 是 |
| 招待费 | 2000元 | 10000元 | 是 |
| 办公用品 | 500元 | 3000元 | 是 |
| 交通费 | 300元 | 2000元 | 否 |
| 通讯费 | 200元 | 500元 | 否 |

超过限额的报销会被标记为"需复核"。

## 注意事项

1. **数据保密**：财务数据敏感，报告中不展示具体个人薪资等隐私信息
2. **金额精度**：金额计算保留2位小数，百分比保留1位小数
3. **异常标记**：超标项目用红色标记，节约项目用绿色标记
4. **审计追溯**：报告包含数据来源和生成时间，便于追溯
